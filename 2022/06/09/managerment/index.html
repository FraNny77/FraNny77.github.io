第一章 管理学绪论

管理的定义

管理就是为了有效地实现组织目标，由管理者利用相关知识、技术和方法对组织活动进行决策、组织、领导并不断创新的过程。

管理的基本职能

决策、组织、领导、控制、创新

  1. 法约尔
    计划、组织、指挥、协调、控制

  2. 古力克和厄威克
    计划、组织、指挥、协调、报告、预算、人事

  3. 哈德罗·孔兹和西里尔·奥康奈里奇
    计划、组织、领导、控制、人事

管理的二重性

指管理的自然属性和社会属性，是马克思主义关于管理问题的基本观点。

生产力 ——> 自然属性——> 管理的一般职能
生产关系 ——> 社会属性——> 管理的特殊职能

自然属性:
1. 质量管理
2. 库房管理
3. 定额管理
4. 成本管理
5. 财务管理

社会属性:
1. 组织目标
2. 组织道德
3. 领导作风
4. 管理理念
5. 人际关系
6. 群体价值观
7. 组织文化

不同社会形态的二重性

资本主义企业管理二重性：
1.是劳动的必要条件，反映了社会化大生产的一般要求；
2.是资本主义价值增值过程进行的管理、监督和调节，体现资本主义生产关系的实质。

社会主义企业管理二重性：
1.合理组织生产力，组织社会化大生产；
2.管理的权力属于劳动者，管理的目的是发展社会主义生产，满足社会全体成员日益增长的需要。

管理者技能

  1. 一般技能：
    技术技能
    人际关系技能
    概念技能

  2. 特殊技能：
    创新能力
    转化能力
    应变能力
    组织协调能力

第二章管理思想史

中国管理思想
  1. 儒家：
    孔子，以仁为核心、以礼为准则，以义来协调；
    孟子，性善论、修身齐家治国平天下；
    荀子，性恶论；
  2. 道家：
    老子，无为而治，让事物按照自身的必然性发展；
  3. 法家：
    韩非，以奖惩的强制手段来求得社会公平，提倡用人唯贤，重农主义，法先王、复古；
  4. 墨家：
    墨子，博爱、节用‘尚贤‘、7
  5. 兵家：
    孙子
  6. 阴阳家：
    平衡、和谐
西方古典管理学思想
  1. 行政管理思想
  2. 生产管理思想
西方管理理论先驱
  1. 亚当·斯密（英国）：《国富论》，劳动分工；
  2. 查尔斯·巴尔奇（英国）：运用数学模型来科学管理；
  3. 罗伯特·欧文（英国，空想社会主义者）：重视人，缩短工时(人事管理)；
西方管理理论的形成与发展

形成：工厂制度的诞生

  1. 机器生产代替手工生产；
  2. 自然力代替人力；
  3. 现代雇佣制度得到确立；
  4. 管理者与被管理者的界限分明；
古典理论

特点
1.基于“经济人”的人性假设；
2.出发点：经济利益驱动；
3.管理手段：“胡萝卜”+“大棒”；
4.研究方法：静态研究管理的一般过程

1. 科学管理思想

泰勒（美国）：19世纪末期，工业化发展，磨洋工现象；
以工人个体为研究对象；
四次实验：泰勒实验、金属切削实验、铁锹实验、搬运实验；

贡献：
  1. 第一次使管理从经验上升为科学；
  2. 讲究效率和调查研究；
  3. 计划和执行相分离；
局限性：
  1. 对工人的看法是错误的；
  2. 仅重视技术，不重视人；
  3. 仅局限于车间管理，内容窄；
2. 行政管理思想

法约尔（法国）：管理过程之父、现代经营管理之父；
以企业组织为研究对象；
首次提出一般管理理论；

企业活动：
  1. 技术性工作；
  2. 商业性工作；
  3. 财务性工作；
  4. 会计性工作；
  5. 安全性工作；
  6. 管理性工作；
法约尔跳板/法约尔桥

减少了不必要的权力距离；

评价
  1. 更具一般性；
  2. 提出管理理论框架；
  3. 原则经过多年证明有效；
3. 一般管理思想

马克斯·韦伯（德国）：以社会组织为研究对象，韦伯的行政管理理论也称为科层制/官僚制。稳定、严密、有效、准确。

三种权威
  1. 传统权威： 依赖于传统或习俗的权利，不利于社会变革，往往非理性、不一致；
  2. 魅力权威： 领导者的使命和愿景激励他人形成的权力基础，合法性基于信念；
  3. 理性法定权威： 以理性和法律规定为基础行使权威。能形成一个客观、具体的组织结构；
韦伯的理性官僚行政组织的特征：
  1. 劳动分工
  2. 职权等级
  3. 正式的选拔
  4. 正式的规则和制度
  5. 非人格性
  6. 职业定向
科层制缺陷
  1. 过分强调机械性地正式组织层面而忽视组织动态面；
  2. 目标错置，例如：过分重视法规而使人员僵化；
  3. 层级节制削弱上级对下级地影响力；
  4. 训练有素地无能；
  5. 永业化使人员丧失斗志；
  6. 升迁按年资事，人员忽视服务对象地利益；
  7. 官僚体制是理想的建构，缺乏实证基础；
霍桑实验
  1. 照明实验（车间照明实验）： 灯光亮度不影响产量；
  2. 福利实验（继电器装配实验）： 梅奥（美国） 参加； 福利不影响产量；后来产量上升原因为：参加实验的光荣感，成员间良好的相互关系；
  3. 访谈实验（大规模的访谈计划）：倾听工人想法，产量大幅提高；
  4. 群体实验（继电器绕线组的工作室实验）：组织内部存在自发形成的非正式群体,这种群体有自己特殊的行为规范，对人的行为起调节和控制作用，加强内部协作关系；
结论
  1. 职工是社会人，生产效率取决于职工的积极性和职工的家庭关系与人际关系；（福利实验）
  2. 企业中存在非正式群体，规范小团体成员的感情倾向和行为；
  3. 新型的领导能力在于提高职工的满足度，管理人员要同时具有经济能力和人际关系能力; (访谈实验)
  4. 存在霍桑效应，对于新环境的好奇和兴趣，导致较佳的成绩，至少在最初阶段如此；（照明实验）

冲击了经济人假设，引发社会人假设。

行为理论

马斯洛的需要层次理论：
生理需要 ——> 安全需要 ——> 社交需要 ——> 自尊需要 ——> 自我实现需要

第三章 管理道德和社会责任
社会责任

社会责任是企业在遵守有关法律、法规和追求经济利益的前提下，还必须承担起追求对社会有益的长期目标的义务，要对整个社会的政治、经济发展负责。

社会义务是企业参与社会的基础。

社会责任和社会反应超出了基本的经济和法律标准。

两种社会责任观

  1. 古典观
    利润：社会活动消耗企业资源
    股东利益：不符合股东利益
    权力：加强企业权力
    责任：非企业家责任
    社会基础：缺乏社会基础
    资源：企业不具备

  2. 社会经济观
    利润：企业参与社会活动增加长期利润
    股东利益：符合股东利益
    权力：企业权力有限
    责任：企业应当承担相应的社会责任
    社会基础：舆论对企业追求社会目标呼声高
    资源：企业拥有承担社会责任所需资源

社会责任具体表现
  1. 对环境
    企业要在保护环境方面发挥主导作用，推动环保技术；
    企业要以“绿色产品”为主要研发对象；
    企业要治理环境；
  2. 对员工
    不歧视员工；
    定期或不定期培训员工；
    营造一个良好的工作环境；
    善待员工；
  3. 对顾客
    提供安全的产品；
    提供正确的产品信息；
    提供售后服务；
    提供必要的指导；
    赋予顾客自主选择权力；
  4. 对竞争对手
    企业不能压制竞争，也不能搞恶意竞争；
  5. 对投资者
    为投资者带来回报；
  6. 对所在地区
    多参与当地社会活动，树立良好的企业形象；
医药企业的社会责任

规范药品生产流通秩序；
企业自主创新能力；
保证国家基本药物的可及性；
诚信经营；

管理道德

特殊的职业道德，管理者行为准则和规范的综合，建立在一般道德准则之上，通过规范管理者的行为去实现调整管理关系的目的，从而提高管理效益。

管理道德规范

忠于职守；
遵纪守法；
实事求是；
团结协作；
尊重人才；

道德观
  1. 道德的功利观（结果主义）：为尽可能多的人提供尽可能多的利益（关注大多数人的利益）；
    例：功利主义，边沁
  2. 道德的权力观（绝对主义）：决策要在尊重和保护个人基本权力的前提下作出（关注所有人的利益）；
  3. 公平理论道德观：要求管理者公平地实施规则；
影响道德的因素
  1. 道德发展阶段；
  2. 个人特征；
  3. 结构变量；
  4. 问题强度；
  5. 组织文化；
道德发展阶段
  1. 前惯例期（并没有道德观念，凡事只会着重个人利益和只为满足自己而行事）
    阶段一（避罚服从取向）： 只单纯为了免受惩罚而服从规范；
    阶段二（相对功利取向）：为得到赞赏（取得利益）；
  2. 惯例期（道德观念是以他人的标准作判断，并以此为方向，尊重法律，对重要人物期望做出反应）
    阶段三（寻求认可取向）；
    阶段四（遵守法规取向）：认为法律至高无上，服从道德规范；
  3. 后惯例期（道德观念超越一般人和社会规范，有自我要求，受个人道德准则规范）
    阶段五（社会法制取向）：相信法律是为了维护社会大众的最大利益，一切会以大众的利益，仍有不足，有时会为了大众利益违法；
    阶段六（普遍伦理取向）：凭自我内心行事，认为有些法律有违其建立的本意；

多数成年人的道德发展处于阶段四；
道德发展可能中断，停留在某个阶段；
人们依次通过这些阶段，不能跨越；

个人特征
  1. 自我强度：度量一个人的信念强度。
    自我强度越高，克制冲动并遵守其信念的可能性越大；
  2. 控制中心：度量主宰自己命运的程度，分内在控制、外在控制；
结构变量

组织的结构设计有助于管理者道德行为的约束，有些提供有利指导，有些约束范围模糊。
例：
上级行为；
合理的绩效评估体系；
报酬的分配方式；
赏罚标准的公平性；

管理道德强度的决定因素
  1. 对受害者伤害多大/对受益者利益多大；
  2. 多少人认为此行为邪恶；
  3. 行为实际发生并造成实际伤害（利益）的可能性多大；
  4. 在行为和其预期后果之间的时间间隔有多长；
  5. 行为的受害者（受益者）与你在 社会上/心理上/身体上 有多近；
  6. 道德行为对有关人员的影响的集中程度如何；
