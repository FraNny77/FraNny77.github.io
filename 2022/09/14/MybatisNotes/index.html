MyBatis

MyBatis是支持普通SQL查询，存储过程和高级映射的优秀持久层框架。MyBatis 消除了几乎所有的JDBC代码和参数的手工设置以及结果集的检索。My Batis使用简单的XML或注解用于配置和原始映射，将接口和Java的POJOs (Plan Old Java Objects,普通的Java对象)映射成数据库中的记录。

Jdbc操作数据库的流程:

  1. 加载驱动: Class.forName(‘驱动的全限定名)
  2. 建立数据库连接: DriverMannager.getConnection() -> Connection
  3. 声明SQL
  4. 创建Statement (编译和发送并执行SQL语句) -> ResultSet (结果集)
  5. 遍历结果集取得数据 -> POJO(JavaBean)
  6. 关闭数据库

入门工程

  1. 添加依赖
  2. 构建sessionFactory -> SqlSession (封装Connection) :建立数据库连接
  3. 创建对象关系映射文件(*.xml): 编写SQL语句，配置对象映射关系文件的路径
  4. 从Mapper.xml文件中创建Statement
  5. 获取结果集, MyBatis帮助我们实现结果集的封装
